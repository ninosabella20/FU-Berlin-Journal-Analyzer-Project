# Reflective Journal Analyzer

Steps to clone and push in the repo:
```
1) Clone the Repository
    git clone <repository-url>

2) Navigate to the Repository Folder
    cd repo-name

3) Check which branch you are working on
    git branch

4) Create and Switch to a New Branch
    git checkout -b <branch-name>

5) Make changes and Add file
    git add <file-name>

    or

    git add .

6) Commit your change
    git commit -m "Your commit message"

7) Push the Branch
    git push origin <branch-name>

```

Steps to create the virtual enviroment 
```
1) Go to the working directory of the project
    cd FU-Berlin-Journal-Analyzer-Project

2) Create the virtual environment
    python -m venv Journal

3) Activate the virtual environment:
    
    Windows: venv\Scripts\activate
    macOS/Linux: source venv/bin/activate

4) Install the requiremnts
    pip install <package-name>

5) To deactivate 
    deactivate

```
